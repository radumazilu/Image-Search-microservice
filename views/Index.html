API Basejump: Image Search Microservice

By radumazilu | See it on GitHub

User Stories:
  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings by requesting /latest.

Example usage:

https://radumazilu-image-search.herokuapp.com/api/imagesearch/taco
https://radumazilu-image-search.herokuapp.com/latest
